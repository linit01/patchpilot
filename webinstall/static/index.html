🗑 Uninstall PatchPilot

Remove PatchPilot from your Kubernetes cluster. This deletes the namespace and all resources within it.

This will permanently delete:

• The patchpilot namespace and all Kubernetes resources
• The PostgreSQL database and all patch history
• The ClusterIssuer for Let's Encrypt
• All PersistentVolumeClaims (PVC data may remain on disk — see post-uninstall note)

⚠ This cannot be undone. Make sure you have a backup if you need the data.

Uninstall reads the namespace from k8s/install-config.yaml. If you've changed it manually, update the config first.
