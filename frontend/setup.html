General Settings

Backup Storage

PatchPilot can back up your database and Ansible inventory. Choose where to store backup archives.

💾 Local Storage
Backups stored in a Docker named volume or local path on the host. Easiest to set up.
🗄️ NFS Share
Backups written to a network share (TrueNAS, Synology, etc). Better for multi-host HA setups.
After setup completes, you'll see the exact snippet to add to your docker-compose.yml or Kubernetes PV manifest.
